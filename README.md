## Requirement(s)

1. Ruby
2. MySQL/MariaDB

## Gem(s)

1. sinatra
2. sinatra-contrib
3. mysql2

## How to Use

1. Clone this repository to your desired location.
   ```bash
   git clone https://github.com/rgxcp/GGIntermediate.git
   ```
2. Change your directory into project.
   ```bash
   cd GGIntermediate
   ```
3. Run the script.
   ```bash
   ruby /path/to/file.rb
   ```
4. Do-what-you-want-with-it!
